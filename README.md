# 🧍 Hangman_JS
A simple Hangman game made with HTML, CSS, and vanilla JavaScript.

## 🗒️ Features
- Customizable grid size
- Keyboard controls (Arrow keys and ZQSD)
- Adjustable game speed
- Score tracking
